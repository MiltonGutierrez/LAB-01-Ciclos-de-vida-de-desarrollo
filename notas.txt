git log -> Historico de los commits que se han hecho. Con los datos: Checksum, autor, fecha, mensaje, rama, etc.
git commit -m -> Guarda los cambios hechos en el repositorio
git status -> Muestra si se han hecho cambios en los archivos que pertenecen al repositorio
git init -> inicia el repositorio en la carpeta en la que se encuentra ubicado
git add -> a√±ade un archivo al repositorio
git remote add origin <url> -> Permite crear un repositorio remoto asociado a uno local, origin esel nombre 
git remote -v -> Permite ver todos los repositorios remotos 
fetch -> Permite traer todas las ramas que se tienen del repositorio 
git push --set-upstream <nombre del repo remoto> <rama> -> permite dejar la rama predeterminadad para poder hacer push
git config --global user.email <email>
git config --global user.name <name>
pwd verificar el directorio donde nos encontramos
git clone <url> copia el repositorio solo trae la rama principal
git push/pull <nombre de repo> <nombre de rama>
